Welcome

HTML is a simple markup language for web browsers.

It defines the structure of a web page and used with CSS and JavaScript to make an interactive and beautiful website.

The Basics

HTML is easy to write. It's made up of tags and attributes.

Tags in HTML are also known as elements. These are the basic buidling blocks of the website.

Most HTML tags are written like this:

<tagname></tagname>

The most important element is the <!DOCTYPE html> element.

This tag tells the browser that the file is a HTML file. The next element is the <html> tag. This tag tells the browser where the actual HTML is.

Most HTML files when starting out will look like this.

<!DOCTYPE html>
<html>
</html>

The two next important tags are the <head> and the <body< tags.

The head tag contains metadata for the HTML file. Including the website's title.

To set up a website's title, we use the <title> tag.

<title>Document</title>

The head tag is also used to link the document to other files. Like CSS and JS files.

But for now, let's focus on body tag. The <p> and the <hn> are used to create paragraphs and header.

The paragraph tag (<p>) simply renders text.

The header tag (<hn>) is used to create a header of n(1-6) level.

Examples.

<h1>I am a header!</h1> <p>I am a paragraph!</p>

Output:

I am a header!

I am a paragraph!

Text can be bolded and italicized using the <b> and <i> tags.

Examples:

<b>BOLD!</b> <i>SLANTED!</i>

Output

BOLD! SLANTED!

Attributes, Links, and Images

Images are placed onto a HTML document using the <img> tag.

But in order to link a image, we use the src attribute. Attributes are more information that allows the browser to know how to format the element.

We use the width and height attribute to resize an image.

Example

<img width="100" height="100" src="https://www.w3.org/StyleSheets/TR/2021/logos/W3C">

Output

Many websites are linked via <a> tags. We set the link we want to visit in the href attribute.

<a href="https://www.wikipedia.org/">Link Text</a>

Output

Link Text

Links and <a> tags are powerful!

Quick Reference

Below is commonly used HTML elements.

Tag Name Use Example
<!DOCTYPE html> Defines the document as a HTML document. <!DOCTYPE html>
<html> Root of HTML document. <html></html>
<head> Contains metadata about the website. <head></head>
<body> Contains the content of the website. <body></body>
<hn> For headers in documents. n is the header level, ranging from 1 to 6. <h1>Hello!</h1>
<p> For paragraphs <p>This is a nice website.</p>
<b> Inline element for bold text. <b>Hi.</b>
<img> Used to insert images. <img src="image.png">
